# Digital Notice Board Management System

## Overview
A console-based Digital Notice Board Management System developed using C++ and Object-Oriented Programming concepts.

## Features
- Admin Login
- Add Notice
- View All Notices
- Search Notice
- View Notices by Category
- Modify Notice
- Delete Specific Notice
- Delete All Notices

## Technologies Used
- C++
- Object-Oriented Programming
- File Handling
- String Manipulation

## Concepts Implemented
- Classes and Objects
- Functions
- File Handling
- Search Operations
- Menu Driven Programming
